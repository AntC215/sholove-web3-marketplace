# ShoLove User Stories

## Marketplace Creator

As a digital asset owner, I want to create a personalized storefront so that I can display and sell my NFTs in a more engaging way.

## Buyer

As a marketplace visitor, I want to browse digital assets visually so that I can discover NFTs in a more interactive experience.

## Wallet User

As a Web3 user, I want to connect my wallet so that I can view, list, and interact with blockchain-based assets.

## NFT Seller

As an NFT seller, I want to list my asset for sale so that other users can purchase it through the marketplace.

## NFT Buyer

As an NFT buyer, I want to view asset details before purchasing so that I can make an informed buying decision.

## Platform User

As a user, I want my marketplace space to feel personal so that my digital assets are displayed in a unique environment.

## Creator

As a creator, I want to customize my island storefront so that my brand and digital assets stand out.

## Collector

As a collector, I want to browse different user islands so that I can discover unique assets and collections.
